# 🛒 Zepto Clone Android App

A modern Zepto-inspired grocery delivery Android application built using Java and Android Studio.

## 🚀 Features

- 📱 Mobile Number Validation
- 🛍️ Product Categories
- 🥦 Fruits & Vegetables
- 🥛 Dairy, Bread & Eggs
- 🍿 Snacks Section
- 🛒 Add To Cart
- ➕ Quantity Increase/Decrease
- ❌ Remove From Cart
- 💳 Checkout & Payment Screen
- 🔔 Notifications
- 📦 Orders Page
- 📍 Address Management
- 📶 Offline Cart Save

## 🛠️ Tech Stack

- Java
- Android Studio
- XML
- RecyclerView
- SharedPreferences
